# YGOFM Fusion Chart

Interactive fusion reference for **Yu-Gi-Oh! Forbidden Memories**.

**Live:** https://rodrigoazlima.github.io/ygofm-app/

## Features

- 204 fusion result cards grouped by type, sorted by ATK
- Hover tooltip with card image and stats
- Search any of the 722 cards by name
- Fusion dialog: **Fuses Into** (what the card produces as material) and **Made From** (what materials produce it)
- Click any result or material in the dialog to chain-navigate

## Data

Card data from [YGO-FM-FusionCalc](https://github.com/Solumin/YGO-FM-FusionCalc) (MIT).  
Card images from [ygoprodeck.com](https://ygoprodeck.com).
